Plano de Estudos – Lógica com JavaScript

Dia 1 – Fundamentos da Lógica e Variáveis

🎯 Objetivo: entender como guardar e manipular dados.

Teoria:

  • O que é lógica de programação.
  • Variáveis (let, const) e tipos de dados (número, texto, booleano).
  • Operadores matemáticos (+, -, *, /, %).
  • O que é lógica de programação.
  • Variáveis (let, const) e tipos de dados (número, texto, booleano).
  • Operadores matemáticos (+, -, *, /, %).

Prática:

  • Criar variáveis com seu nome, idade e cidade.
  • Somar dois números e mostrar no console.
  • Calcular o resto da divisão de 10 por 3.

Desafio:

  • Criar um código que diga “Seu nome é X e você tem Y anos”.

Dia 2 – Condições (if, else, else if)

🎯 Objetivo: tomar decisões no código.

Teoria:

  • Estrutura do if e else.
  • Operadores de comparação (==, ===, !=, >, <, >=, <=).
  • Operadores lógicos (&&, ||, !).

Prática:

  • Criar um programa que verifica se uma pessoa é maior de idade.
  • Verificar se um número é par ou ímpar.

Desafio:

  • Criar um código que:
  • Pergunta a nota de um aluno.
  • Diz “Aprovado” se >= 6, senão “Reprovado”.

Dia 3 – Loops (for e while)

🎯 Objetivo: repetir ações.

Teoria:

  • Estrutura do for e while.
  • Contadores e incremento (i++).

Prática:

  • Contar de 1 a 10 no console.
  • Somar números de 1 a 100.

Desafio:

  • Criar um código que mostre a tabuada de um número.

Dia 4 – Funções

🎯 Objetivo: criar blocos de código reutilizáveis.

Teoria:

  • Como criar funções (function).
  • Parâmetros e retorno (return).

Prática:

  • Criar uma função que soma dois números.
  • Criar uma função que verifica se um número é par.

Desafio:

  • Criar uma função que recebe um nome e retorna “Olá, [nome]!”.

Dia 5 – Arrays e Objetos

🎯 Objetivo: guardar e acessar listas de dados.

Teoria:

  • Arrays ([]) e seus métodos básicos (push, pop, length).
  • Objetos ({}) e propriedades.

Prática:

  • Criar um array com 5 frutas e mostrar cada uma.
  • Criar um objeto representando uma pessoa.

Desafio:

  • Criar uma lista de alunos com nome e nota, e mostrar quem foi aprovado.

Dia 6 – Eventos e DOM

🎯 Objetivo: interagir com a página.

Teoria:

  • O que é DOM.
  • Como selecionar elementos (document.querySelector).
  • Eventos (click, input).

Prática:

  • Criar um botão que mostra “Você clicou!” no console.
  • Criar um input que mostra o valor digitado.

Desafio:

  • Criar um campo para digitar um nome e um botão que exibe “Olá, [nome]!” na página.

Dia 7 – Mini Projeto Final

🎯 Objetivo: juntar tudo o que aprendeu.

Projeto sugerido:

  • Jogo de Adivinhação:
  • Gerar um número aleatório de 1 a 10.
  • Usuário digita um número.
  • Mostrar se acertou ou errou.

Extras:

  • Criar um “contador regressivo” para um evento.
  • Fazer um “calculador de idade” com base no ano de nascimento.

💡 Dicas para o estudo:

  • Treine todo dia, mesmo que por pouco tempo.
  • Use o console.log() para entender o que está acontecendo.
  • Teste ideias diferentes além dos exemplos.
  • Erros fazem parte do aprendizado — leia a mensagem de erro e tente entender.
